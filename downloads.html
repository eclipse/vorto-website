Prerequisites

  • Eclipse Kepler (Standard) or Eclipse Luna (Standard)
  • Xtext 2.7.x

Vorto Update Site

Install Vorto using the Standard Eclipse Update mechanism:
Choose Help -> Install New Software... from the menu bar and Add... Insert the update site URL Vorto Update Site Select the components of your choice and complete the wizard by clicking the Next button until you can click Finish. After a quick download and a restart of Eclipse, Vorto is ready to be used.
