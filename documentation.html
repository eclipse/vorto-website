Getting Started

Installing the Vorto Toolset

Prerequisites:

  • Eclipse Kepler or Eclipse Luna
  • JDK 1.7
  • Xtext 2.7.* redistributable (http://download.eclipse.org/modeling/tmf/xtext/updates/composite/releases/)

  1. From your IDE, click on menu "Help -> Install New Software..."
  2. Enter the Update site https://eclipse.org/vorto/update and hit enter (It may take a few seconds until you can see the list of available software).
    3. Select the Vorto components from the software list. Click 'Next' and verify the installation details. Click continue
  3. A software agreement and license page is shown. Select "I accept the terms of the license agreements" and click Finish
  4. Restart the IDE

Describe devices with Information Models

The Vorto Toolset provides two types of Editors to describe the semantics of of devices. A Function Block Editor describes abstract properties, such as status, fault, configuration and events as well as operations to change the behaviour of a device. An Information Model Editor aggregates various function blocks and is enriched with vendor specific meta data.

Integrate Devices into target platforms

Information Models can easily be integrated into a any platform upon the availability of so - called platform-specific code generators. These generators understand the Information Meta Model and platform specific meta data attached to the model and translates it into platform specific code, e.g. in the form of Java, XML, C++ etc.

Manage and share Information Models

The Information Model Repository is a server based technology which shall be used as the centralized storage location for information models. As soon as Release 1.0 is available the repository will be hosted by the Eclipse Foundation.
